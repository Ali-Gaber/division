# Division

A simple to use yet powerfull styling oriented package with syntax inspired by CSS.

## Getting Started

**A basic example
```dart
Parent()
    .width(200)
    .height(250)
    .backgroundColor(Colors.grey)
    .elevation(5)
    .borderRadius(all: 30)
    .animate(500, Curves.easeIn)
    .build()

Txt('Some text')
    .bold()
    .fontSize(24)
    .width(100)
    .height(150)
    .backgroundColor(Colors.grey)
    .ripple(true)
    .build()

```
